# Media Card Wrangler — Releases

Public download host for **Media Card Wrangler**, a macOS app that safely
flattens copied camera-card folders for editorial.

All builds here are signed with a Developer ID and notarized by Apple.

- **Download the latest version:** https://mediacardwrangler.com/download
- **All versions (including previous releases):** see the [Releases](../../releases) tab.

The app's source code is kept in a separate private repository; this repo
exists only to host the release binaries so they have stable public download
URLs.
